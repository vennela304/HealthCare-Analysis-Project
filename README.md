# HealthCare-Analysis(Interactive Dashboard Creation Using MS Excel)
## Objectives
•	Analyze patient admissions and treatment data
•	Identify trends across departments and time periods
•	Improve hospital resource utilization insights
•	Build interactive dashboards for better reporting
•	Support data-driven decision making in healthcare

## Dataset Used
- <a href="https://github.com/vennela304/HealthCare-Analysis-Project/blob/main/Healthcare%20Analysis.xlsx">Healthcare Data</a>

## Question KPI'S
1.	Total Patients
2.	Total Doctors
3.	Total Visits
4.	Average Age of Patients
5.	Top 5 Diagnosed Conditions
6.	Follow-Up Rate
7.	Treatment Cost Per Visit 
8.	Total Lab Tests Conducted
9.	Percentage of Abnormal Lab Results
10.	Doctor Workload (Avg. Patients Per Doctor)

## Process 
1️⃣ Data Collection
•	Collected healthcare-related datasets containing patient, department, treatment, and billing information.
•	Ensured the dataset was anonymized/simulated to maintain data privacy.

2️⃣ Data Cleaning & Preprocessing
•	Used Excel to clean raw data by removing duplicates and handling missing values.
•	Standardized column names, date formats, and categorical values for consistency.
•	Validated data accuracy before loading it into the database.

4️⃣ Data Loading
•	Imported cleaned data into MySQL tables.
•	Verified data integrity using validation queries.

5️⃣ SQL Analysis
•	Wrote SQL queries using joins, subqueries, aggregations, and filters.
•	Created views and stored procedures for reusable analysis.
•	Extracted key healthcare metrics such as patient admission trends and department performance.

6️⃣ Data Visualization
•	Connected MySQL data to Power BI and Tableau.
•	Created interactive dashboards with filters and KPIs.
•	Visualized trends in patient flow, treatment outcomes, and resource utilization.

7️⃣ Insights & Reporting
•	Interpreted dashboard results to identify key healthcare insights.
•	Documented findings to support data-driven decision making.

8️⃣ Validation & Review
•	Cross-verified dashboard results with SQL query outputs.
•	Ensured consistency across Excel, SQL, Power BI, and Tableau results.

## Key Insights

•	Certain departments handle a higher patient load compared to others
•	Seasonal trends impact patient admissions
•	Data visualization helps identify performance gaps and resource needs

## Final Conclusion

This Healthcare Data Analysis project successfully demonstrates how raw healthcare data can be transformed into meaningful insights using a structured data analytics approach. By leveraging SQL for data querying, Excel for data cleaning, and Power BI and Tableau for visualization, the project provides a comprehensive view of patient trends, department performance, and hospital resource utilization.
The interactive dashboards enable quick identification of patterns and support data-driven decision making, helping healthcare organizations improve operational efficiency and patient care. This project highlights strong skills in database management, data analysis, and business intelligence, and showcases the practical application of analytical tools in a real-world healthcare scenario.
